## Description:
This project is the backend application for an online store, built using Java and Spring Boot. It handles HTTP requests, manages user sessions with JWT, provides authentication and authorization and connects to a MySQL database hosted on an external Amazon RDS server.

## Technologies Used:
- Java
- Spring Boot
- MySQL
- Amazon RDS
- Maven
- Swagger

## Getting Started:
Clone this repository to your local machine.
Configure the application properties to connect to your MySQL database hosted on Amazon RDS.
Build and run the application using Maven.

## Configuration:
Ensure that you have Java and Maven installed on your system.
Update the application.properties file with your MySQL database connection details.
